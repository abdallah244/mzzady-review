## 💵 الإيداع (Deposit)

### طرق الإيداع المتاحة

| الطريقة        | المدة    | الرسوم |
| -------------- | -------- | ------ |
| 📱 InstaPay    | فوري\*   | 0%     |
| 🏦 تحويل بنكي  | 1-3 أيام | 0%     |
| 💳 فودافون كاش | فوري\*   | 0%     |

\*بعد موافقة الأدمن

### خطوات الإيداع

```
┌────────────────────────────────────────────────────────┐
│  1️⃣ المستخدم يختار طريقة الإيداع                       │
│         ↓                                              │
│  2️⃣ يُدخل المبلغ المراد إيداعه                         │
│         ↓                                              │
│  3️⃣ يحوّل المبلغ للحساب المحدد                         │
│         ↓                                              │
│  4️⃣ يرفع إثبات التحويل (صورة/سكرين شوت)               │
│         ↓                                              │
│  5️⃣ الطلب في حالة "انتظار"                            │
│         ↓                                              │
│  6️⃣ الأدمن يراجع ويوافق/يرفض                          │
│         ↓                                              │
│  7️⃣ ✅ موافقة: يُضاف للرصيد                           │
│     ❌ رفض: إشعار للمستخدم                             │
└────────────────────────────────────────────────────────┘
```

### حالات طلب الإيداع

| الحالة     | الوصف              | الأيقونة |
| ---------- | ------------------ | -------- |
| `pending`  | في انتظار المراجعة | 🟡       |
| `approved` | تمت الموافقة       | ✅       |
| `rejected` | مرفوض              | ❌       |
