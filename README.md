## 🎯 نظرة سريعة على المشروع

**مزادي (Mazzady)** منصة مزادات إلكترونية متكاملة تتيح:

- 🛒 **المشاركة في المزادات** - مزايدة على المنتجات
- 💼 **بيع المنتجات** - إنشاء مزاداتك الخاصة
- 💳 **محفظة آمنة** - إيداع وسحب الأموال
- ⭐ **تقييم البائعين** - نظام إعجابات وتقييمات
- 🔔 **إشعارات فورية** - تنبيهات للمزايدات
