## 📋 نظرة عامة

نظام المزادات في مزادي يدعم عدة أنواع من المزادات لتلبية احتياجات مختلفة:

```
┌─────────────────────────────────────────────────────────┐
│                   🏆 أنواع المزادات                     │
├─────────────────────────────────────────────────────────┤
│  🔥 مزاد عادي      │  المزاد التقليدي - أعلى مزايد يفوز  │
│  ⚡ مزاد سريع      │  Flash Auction - مدة قصيرة          │
│  👥 مزاد جماعي     │  Group Auction - مشاركة جماعية      │
│  🔒 مزاد خاص      │  Private Auction - بدعوة فقط       │
│  🔄 مزاد عكسي     │  Reverse Auction - أقل سعر يفوز    │
└─────────────────────────────────────────────────────────┘
```

---

## 🔥 المزاد العادي (Standard Auction)

### كيف يعمل؟

```
┌────────────────────────────────────────────────────────┐
│  📦 البائع ينشئ المزاد                                 │
│         ↓                                              │
│  ⏳ المزاد في حالة انتظار (Pending)                    │
│         ↓                                              │
│  ✅ الأدمن يفعّل المزاد                                │
│         ↓                                              │
│  🏃 المزاد نشط (Active) - المزايدة مفتوحة              │
│         ↓                                              │
│  ⏰ انتهاء الوقت                                       │
│         ↓                                              │
│  🏆 الفائز = صاحب أعلى مزايدة                          │
└────────────────────────────────────────────────────────┘
```

### إنشاء مزاد جديد

| الحقل               | الوصف                           | مطلوب |
| ------------------- | ------------------------------- | ----- |
| `productName`       | اسم المنتج                      | ✅    |
| `startingPrice`     | السعر المبدئي                   | ✅    |
| `minBidIncrement`   | أقل زيادة للمزايدة (افتراضي: 1) | ❌    |
| `durationInSeconds` | مدة المزاد (1 ثانية - 7 أيام)   | ✅    |
| `category`          | الفئة                           | ✅    |
| `mainImage`         | الصورة الرئيسية                 | ✅    |
| `additionalImages`  | صور إضافية (حتى 9)              | ❌    |
| `isFeatured`        | إضافة لأبرز المزادات            | ❌    |

### حالات المزاد

| الحالة       | الوصف             | الإجراءات المتاحة |
| ------------ | ----------------- | ----------------- |
| 🟡 `pending` | في انتظار التفعيل | تعديل، حذف، تفعيل |
| 🟢 `active`  | المزاد نشط        | مزايدة، مشاهدة    |
| 🔴 `ended`   | المزاد انتهى      | عرض النتائج       |

## ⚡ المزاد السريع (Flash Auction)

### المميزات

- ⏱️ **مدة قصيرة**: عادة من دقائق لساعات
- 🔥 **حماس عالي**: المنافسة سريعة
- 💰 **صفقات مميزة**: عادة بأسعار أقل

### الفرق عن المزاد العادي

```
المزاد العادي          المزاد السريع
─────────────          ─────────────
مدة: 1-7 أيام          مدة: دقائق-ساعات
تخطيط مسبق             قرارات سريعة
مزايدات متأنية          مزايدات فورية
```

## 👥 المزاد الجماعي (Group Auction)

### كيف يعمل؟

```
┌────────────────────────────────────────────────────────┐
│  1️⃣ البائع يحدد عدد المشاركين المطلوب                  │
│         ↓                                              │
│  2️⃣ المستخدمون ينضمون للمزاد                          │
│         ↓                                              │
│  3️⃣ عند اكتمال العدد → يبدأ المزاد                    │
│         ↓                                              │
│  4️⃣ المشاركون يتنافسون                                │
│         ↓                                              │
│  🏆 الفائز من بين المشاركين فقط                        │
└────────────────────────────────────────────────────────┘
```

---

## 🔒 المزاد الخاص (Private Auction)

### المميزات

- 🔐 **بدعوة فقط**: روابط خاصة للمشاركة
- 👥 **مجموعة محددة**: للأصدقاء أو العملاء المميزين
- 🎁 **صفقات حصرية**: عروض خاصة

### كيفية الانضمام

```
البائع ينشئ مزاد خاص
        ↓
يحصل على رابط دعوة فريد
        ↓
يشارك الرابط مع من يريد
        ↓
المدعوون فقط يستطيعون المزايدة
```

## 🔄 المزاد العكسي (Reverse Auction)

### كيف يختلف؟

| المزاد العادي    | المزاد العكسي    |
| ---------------- | ---------------- |
| أعلى مزايدة تفوز | أقل مزايدة تفوز  |
| السعر يرتفع      | السعر ينخفض      |
| للشراء           | للخدمات/المشاريع |

### حالات الاستخدام

- 🏗️ مشاريع البناء
- 🚚 خدمات الشحن
- 💼 الخدمات المهنية

## 🤖 المزايدة التلقائية (Auto-Bid)

### كيف تعمل؟

```
┌────────────────────────────────────────────────────────┐
│  المستخدم يحدد:                                        │
│  • الحد الأقصى للمزايدة: 1000 جنيه                     │
│  • الزيادة التلقائية: 10 جنيه                          │
├────────────────────────────────────────────────────────┤
│  النظام يزايد تلقائياً:                                │
│                                                        │
│  مزايدة جديدة: 500 ← النظام يزايد: 510                 │
│  مزايدة جديدة: 520 ← النظام يزايد: 530                 │
│  مزايدة جديدة: 990 ← النظام يزايد: 1000               │
│  مزايدة جديدة: 1010 ← ⛔ توقف (وصل للحد)              │
└────────────────────────────────────────────────────────┘
```

### إعداد المزايدة التلقائية

| الإعداد        | الوصف                   |
| -------------- | ----------------------- |
| `maxBidAmount` | الحد الأقصى الذي ستدفعه |
| `bidIncrement` | مقدار الزيادة في كل مرة |

## 📈 قواعد المزايدة

### الحد الأدنى للمزايدة

```
المزايدة الجديدة ≥ أعلى مزايدة + الحد الأدنى للزيادة

مثال:
أعلى مزايدة حالية: 500 جنيه
الحد الأدنى للزيادة: 10 جنيه
─────────────────────────────
أقل مزايدة مقبولة: 510 جنيه
```

### التحقق من الرصيد

```
قبل المزايدة:
رصيد المحفظة ≥ قيمة المزايدة
        ↓
    ✅ مزايدة مقبولة
    ❌ رصيد غير كافٍ
```
